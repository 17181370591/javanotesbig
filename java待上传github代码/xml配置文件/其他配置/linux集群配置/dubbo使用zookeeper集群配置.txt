Dubbox连接zookeeper集群
修改服务提供者和服务调用者的spring 配置文件

<dubbo:registry protocol="zookeeper" address="${dubbo.address}"/>
dubbo.address=${my.dubbo.address}
my.dubbo.address=192.168.153.130\:2181,192.168.153.131\:2181,192.168.153.131\:2182

或者

<dubbo:registry protocol="zookeeper" address="192.168.153.130:2181,192.168.153.131:2181,192.168.153.131:2182"/>
